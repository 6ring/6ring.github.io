Join Us Now!

To join the ring, two things have to be done: First, adding your site to the member list found at the end of this Javascript file in our GitHub repository.

Second, adding links to the previous and next members of the ring 

https://6ring.github.io/prev?[your-site-id]
https://6ring.github.io/next?[your-site-id]

to your site as well as a link to the webring navigation 

https://6ring.github.io/?[your-site-id]

It is recommended that you also place the following script in all of your pages' headers 

<script type="text/javascript" data-sixring-id="[your-site-id]"
        src="https://6ring.github.io/6ring.js"></script>

This will enable better integration with the ring navigation otherwise prohibited by the same-origin policy, and provides a widget you can use instead of manual links. To do that, insert 

<script type="text/javascript">document.write(SIXRING_WIDGET)</script>

where you want the widget to appear.
