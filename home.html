Join Us Now!

To join the ring, two things have to be done: First, add your site to the member list found at the end of this Javascript file in our GitHub repository.

Second, add one or more of the following links to your homepage: 

https://6ring.github.io/?[your-site-id]
https://6ring.github.io/prev?[your-site-id]
https://6ring.github.io/next?[your-site-id]
https://6ring.github.io/random?[your-site-id]

It is recommended that you also place the following script in all of your pages' headers 

<script type="text/javascript" data-sixring-id="[your-site-id]"
        src="https://6ring.github.io/6ring.js"></script>

This will enable better integration with the ring navigation otherwise prohibited by the same-origin policy, and provides a widget

you can use instead of manual links: Just insert 

<script type="text/javascript">document.write(SIXRING_WIDGET)</script>

where you want the widget to appear!
