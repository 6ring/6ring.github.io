Join Us Now!

To join the ring, all that has to be done is adding your site to the list found in this Javascript file in our GitHub repository.

To improve discoverability, it would help if you could embed the following links in your site: 

https://6ring.github.io/?<your-site-id>
https://6ring.github.io/prev?<your-site-id>
https://6ring.github.io/next?<your-site-id>

You might also want to add the following code to all of your pages: 

<script src="https://6ring.github.io/6ring.js" type="text/javascript"></script>

This will enable better integration with the ring navigation, otherwise prohibited by the same-origin policy. Some further details can be found in our README file.
